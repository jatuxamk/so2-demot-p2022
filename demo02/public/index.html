Demo 2: Web Service -perusteita

Tämä on demo Web Service-tyylisestä palvelinsovelluksesta, jonka tarkoituksen on keskutella toisen sovelluksen kanssa json-muodossa.

Käytä demoa kirjoittamalla osoiteriville urlin loppuun /kayttajat, niin saat käyttäjälistauksen json-muodossa. Yksittäisen käyttäjän tiedot saat reitillä /kayttajat/id, jossa "id" on halutun käyttäjän id, esim. 7.
